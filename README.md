### GitHub Pages:
 https://alinashamina.github.io/ahj-10-geolocation-media/

### Описание
Приложение "Timeline" позволяет добавлять текстовые посты, которые привязываются к геолокации пользователя.  
При невозможности получить координаты появляется модальное окно для ручного ввода.
